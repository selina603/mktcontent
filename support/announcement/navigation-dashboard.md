# Azure 官网信息导航栏和服务仪表盘更新
更新时间：2016年2月3日

自从在中国大陆正式商用至今，Azure 陆续增加了很多新的服务。为了让大家更轻松地根据需求按图索骥，了解每个服务的用途和作用，我们近期重新调整了 Azure 官网上的信息导航栏和服务仪表盘上面的 Azure 服务分类。调整后的分类面向用户的业务场景，与服务功能描述更加契合，以便您更快速地了解不同服务的用途。

自2016年2月3日起正式使用新分类，详细分类如下：
